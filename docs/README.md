## üìä Project Summary

**KnoAD-Net** is a multi-modal deep learning system that combines MRI brain imaging with clinical data to detect Alzheimer's Disease. Key innovations:

‚úÖ Vision Transformer (ViT) for MRI analysis  
‚úÖ Cross-modal attention for imaging + clinical data fusion  
‚úÖ Retrieval-Augmented Generation (RAG) for explainable predictions  
‚úÖ Optimized for CPU training (AMD Ryzen 5000)  
‚úÖ Three-class classification: CN / MCI / AD
